Detailed Description

Custom class for radio buttons.

Objects of this class are radio buttons that have associated an integer number representing a player (0: first player; 1: second player; etc.). This index is used to check for which player the number of held cards has been specified, i.e., in which groupboxs of the NumberOfCardsForEachPlayerWindow instance (where there is a groupbox of radio buttons, more precisely of CustomRadioButton instances, for each each player) a groupbox of radio button) a radio button has been checked.
